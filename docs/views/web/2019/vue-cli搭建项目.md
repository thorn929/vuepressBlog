#  使用vue-cli脚手架指南



## 下载vue-cli脚手架

[vue-cli官网](<https://cli.vuejs.org/zh/>)

```
npm install -g @vue/cli
```

### 查看一个vue版本

```
vue --version
```

**3.x的版本可以直接使用`vue ui`2.x的版本不能使用vue ui等功能建议下载vue3.x的版本**

# 创建项目

打开控制台`cd`到你工作的文件夹

```
vue ui
```

* 页面会自动跳转一个http://localhost:8000/project/select
